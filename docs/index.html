فونت آرا افزونه ای رایگان و متن باز

نسخه جدید ۴.۳.۰ افزونه فونت آرا منتشر شد.

فونت آرا، افزونه‌ای رایگان و متن‌باز برای بهبود فونت فارسی در تمام وب

اگر از فونت‌های پیش‌فرض و ناخوانای سایت‌ها و شبکه‌های اجتماعی کلافه شدید با استفاده از افزونه‌ی متن‌باز فونت‌آرا می‌توانید تمام سایت‌ها را با فونت دلخواه‌تان یا گلچینی از بهترین فونت‌های فارسی مشاهده نمایید. همچنین مشکل سایت‌هایی مانند Github, Goodreads, Medium و Slack را که پشتیبانی مناسبی از زبان‌های راست به چپ ندارند، نخواهید داشت.

فونت‌آرا را برای مرورگر خود دریافت کنید:

تغییرات نسخه جدید ۴.۳.۰

این نسخه روی پایدارتر شدن تجربه‌ی آپدیت، گسترش پشتیبانی اختصاصی سایت‌ها، بهتر شدن مسیر فونت سفارشی و مرتب‌تر شدن رابط کاربری تمرکز دارد.


پشتیبانی بهتر از سایت‌ها
  • افزوده شدن یا بهبود CSS اختصاصی برای سایت‌های هوش مصنوعی و ابزارهای پرکاربرد مانند ChatGPT، Claude، Gemini، Copilot، Perplexity، Poe، OpenRouter، DeepSeek، Qwen، NotebookLM، AI Studio و Arena.
  • بهبود پشتیبانی اختصاصی برای Google، Gmail، GitHub، Instagram، Facebook، LinkedIn، X، YouTube، Telegram، WhatsApp، Slack، Trello، Wikipedia و DuckDuckGo.
  • به‌روزرسانی ترتیب و دسته‌بندی سایت‌های محبوب بر اساس اهمیت و کاربرد، جایگزینی Twitter با X و حذف WordPress از فهرست پیش‌فرض.
فونت سفارشی و پایداری آپدیت
  • بهبود مهاجرت داده‌های ذخیره‌شده تا تنظیمات سایت‌های پیش‌فرض، آیکن‌ها و CSS اختصاصی بعد از آپدیت همگام شوند و کاربر کمتر نیاز به نصب دوباره داشته باشد.
  • ایمن‌تر شدن پردازش فونت‌های سفارشی، نرمال‌سازی MIME در data URLها، تشخیص بهتر فرمت فایل و حذف رکوردهای نامعتبر.
  • اصلاح fallback فونت انتخاب‌شده وقتی فونت سفارشی حذف یا نامعتبر می‌شود، برای جلوگیری از گیر کردن افزونه روی حالت خراب.
رابط کاربری و آیکن‌ها
  • جایگزینی آیکن‌های دستی با لوگوهای رسمی‌تر و یک‌دست‌تر برای سرویس‌های جدید، به‌همراه آیکن اختصاصی OpenRouter و X.
  • انتقال دکمه‌ی افزودن فونت دلخواه کنار انتخاب فونت، کوچک‌تر و خنثی‌تر شدن badge نسخه، سبز شدن کلید روشن/خاموش و فاصله‌گذاری بهتر بخش انتخاب فونت از آیکن‌ها.
  • کامنت شدن حمایت از FontAra و تمیزتر شدن بخش پایینی popup.
زیرساخت و کیفیت انتشار
  • افزوده شدن ساخت source package برای Firefox و بهبود ابزارهای build و release افزونه‌های MV3.
  • حذف تست‌های شکننده‌ی قرارداد CSS و فایل‌های قدیمی وابسته به selectorهای پرتغییر، هم‌زمان با حفظ تست‌های کاربردی‌تر برای storage، custom font، manifest، سایت‌ها و UI.
  • پاک‌سازی کدها و permissionهای غیرضروری، از جمله حذف مسیر refresh خودکار tabها طبق تصمیم جدید محصول.

تغییرات نسخه جدید ۴.۲.۱

این نسخه شامل تغییرات مهم از بازنویسی زیرساخت افزونه تا بهبود عملکرد، پایداری و پشتیبانی بهتر از ادیتورهای پیچیده است.


بهبودهای اصلی
  • جایگزینی ساختار Plasmo با یک ساختار native برای ساخت افزونه و آماده‌سازی بهتر نسخه‌های Chrome، Firefox، Edge، Brave، Opera و Safari.
  • بهبود الگوریتم اعمال فونت روی صفحه‌ها، کاهش پردازش‌های تکراری و سبک‌تر شدن اجرای افزونه روی سایت‌های سنگین.
  • رفع مشکل تغییر فونت در ادیتورهای contenteditable مثل ورودی ChatGPT و جلوگیری از حلقه‌های تکراری بین افزونه و خود سایت.
  • بهبود اختصاصی ChatGPT با اعمال فونت از طریق CSS سایت به جای دست‌کاری مداوم نودهای در حال استریم، برای کاهش لگ هنگام دریافت پاسخ.
  • اعمال هوشمند فونت روی ادیتورهای قابل‌ویرایش با کمک استایل محاسبه‌شده‌ی همان المنت، بدون تغییر مستقیم روی خود متن و بدون دست‌کاری غیرضروری DOM.
  • بهبود واکنش افزونه به تغییرات داینامیک صفحه، تغییر فونت انتخاب‌شده و فونت‌های سفارشی.
فونت‌ها و فونت‌های سفارشی
  • افزوده شدن فونت آراد به فهرست فونت‌های داخلی فونت‌آرا.
  • افزوده شدن فونت آذرمهر به‌عنوان asset داخلی و بهبود بارگذاری فونت‌های bundled با font-display: swap.
  • بهبود تولید امن‌تر @font-face برای فونت‌های سفارشی و جلوگیری از ورود داده‌های نامعتبر.
  • بازنشانی خودکار فونت انتخاب‌شده بعد از حذف فونت سفارشی‌ای که در حال استفاده بوده است.
رابط کاربری و تجربه استفاده
  • پایدارتر شدن پنجره‌ی افزونه و صفحه‌ی تنظیمات، از جمله مدیریت خطاها، فوکوس Drawer و باز شدن صفحه‌ی تنظیمات.
  • نمایش درست‌تر پیش‌نمایش فونت‌ها و اعمال فونت انتخاب‌شده در خود رابط کاربری افزونه.
  • اصلاح مسیر و وضعیت آیکن‌های افزونه در حالت‌های فعال و غیرفعال.
  • بهبود mount شدن رابط React در popup و options و استفاده از runtime production برای build نهایی.
زیرساخت و کیفیت انتشار
  • ارتقای React به نسخه ۱۹، Tailwind CSS به نسخه ۴ و به‌روزرسانی ابزارهای Node و وابستگی‌های کم‌ریسک.
  • افزودن Biome برای lint و format و اضافه شدن اسکریپت‌های بیشتر برای بررسی، تست و ساخت نسخه نهایی.
  • سخت‌گیرانه‌تر شدن بررسی‌های manifest، storage، workflowهای انتشار و تنظیمات امنیتی افزونه.
  • افزایش تست‌های واحد برای مسیرهای مهم مثل storage، font injection، سایت‌های محبوب، فونت‌های سفارشی، آیکن‌ها و رابط کاربری.

حمایت از فونت آرا

سلام دوستان عزیز! 👋

خیلی خوشحالم که از فونت آرا استفاده می‌کنید. این پروژه با عشق و برای بهبود تجربه وب فارسی ساخته شده و هدفش زیباتر کردن محتوای فارسی روی اینترنت بوده.

اول از همه باید بگم که این نسخه جدید فونت آرا با حمایت و تلاش تیم موچت (محصول دیگر من که برای ساخت چت‌بات‌های هوشمند پشتیبانی و فروش ساختم) پیاده‌سازی شده و از همه همکاران عزیزم که در این مسیر کمک کردند سپاسگزارم.

اگر دوست دارید از فونت آرا حمایت کنید، ساده‌ترین راه اینه که اون رو به دوستانتون معرفی کنید. میتونید این توییت یا این پست لینکدین رو به اشتراک بگذارید یا به هر طوری که دوست دارید فونت آرا به دوستانتون معرفی کنید .

همین معرفی‌های شما بزرگ‌ترین هدیه به فونت آرا هست و باعث میشه انگیزه بیشتری برای بهبود و توسعه داشته باشیم.
